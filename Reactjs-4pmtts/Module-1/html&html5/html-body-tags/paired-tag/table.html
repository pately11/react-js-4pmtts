Table tag is used to give our information in tabular form.

<tr> tag is used for insert row in the table. <td>tag is used for insert data in the table data. <th> tag is used to add heading in the table.
  • rowspan is used to merge two or more rows in the table.
  • colspan is used to merge two or more columns in the table.
  • cell spacing is used to give space between each cell to the outside.
  • cell padding is used to give space between content inside the cell.
  • border attribute is used to give border to the tbale.

    • Example:-

    Name Age
    Jill Smith 43
    Eve Jackson 57
