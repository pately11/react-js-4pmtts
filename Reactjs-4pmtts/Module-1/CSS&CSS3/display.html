The display property specifies the display behavior or the type of rendering box of an element.

  • Default value of display property is inline.

    • Example:-

    The display Property

    display: none:

    Lorem ipsum dolor....

    HELLO WORLD!

    Vestibulum volutpat tellus diam....

    display: inline:

    Lorem ipsum dolor....

    HELLO WORLD!

    Vestibulum volutpat tellus diam....

    display: block:

    Lorem ipsum dolor....

    HELLO WORLD!

    Vestibulum volutpat tellus diam....

    display: inline-block:

    Lorem ipsum dolor....

    HELLO WORLD!

    Vestibulum volutpat tellus diam....
