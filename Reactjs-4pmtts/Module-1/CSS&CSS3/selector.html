    Sibling Selector

    There are the two sibling selector.

    1.Adjacent sibling selector.

    2.General sibling selector.

    Adjacent Sibling Selector is select by using the '+'.

    General Sibling Selector is select by using the '~'.

    Adjacent sibling selector only select the first child.

    General sibling selector only select the every child.
