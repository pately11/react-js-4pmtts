Que-8 How to check version of React Js?

The package.json file in your project directory contains the version information for all the dependencies, including React.

  • Open the package.json file in the root directory of your React project.
  • Look for the react and react-dom entries under the dependencies section.

    • Ex:

            {
            "dependencies": {
              "react": "^18.2.0",
              "react-dom": "^18.2.0",
              // other dependencies
            }
          }
    The version number next to react and react-dom indicates the version of React.js that is installed.
