What is Redux ?

Ans :

Redux is a tool used in React apps to manage and organize app data (state) in one central place. It helps keep data consistent and easy to debug, especially in big apps.

Why is it used in React applications?

Ans :

  • Central place for all app data.
  • Predictable state changes.
  • Easy to debug with tools like Redux DevTools.
  • Simplifies sharing data between components.
