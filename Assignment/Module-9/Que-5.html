Que-5 What is Components in React Js?

n React.js, components are the fundamental building blocks used to create user interfaces. A React application is typically made up of multiple components, each responsible for rendering a small, reusable piece of the UI. Components in React help in organizing the UI into independent, isolated, and reusable parts, making it easier to build and maintain complex applications.

Types of components:

  • Functional Components:These are the most common type of components in modern React. They are simply JavaScript functions that accept props (properties) as an argument and return a React element (usually JSX).

    • Example:

    function Greeting(props) {
    return <h1>Hello, {props.name}!</h1>;
  }
  • Class Components:These were more commonly used in earlier versions of React but are less common now due to the introduction of hooks in functional components. Class components are JavaScript classes that extend React.Component and must have a render() method that returns JSX.

    • Example:

    class Greeting extends React.Component {
    render() {
      return <h1>Hello, {this.props.name}!</h1>;
    }
  }
