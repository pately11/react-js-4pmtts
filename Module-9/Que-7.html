Que-7 How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?

Installing React.js on Windows and Linux involves setting up Node.js, which includes NPM (Node Package Manager). Below are the steps for installing React.js, NPM, and checking their versions on both operating systems.

1. Installing Node.js and NPM on Windows

Step 1: Download Node.js Installer

  • Visit the Node.js official website.
  • Download the Windows installer

    • Step 2: Install Node.js

  • Run the downloaded installer.
  • Follow the installation prompts, ensuring that the checkbox for installing NPM is selected.
  • The installer will install both Node.js and NPM.

    • Step 3: Verify Installation

  • Open the Command Prompt (cmd) or PowerShell.
  • Run the following commands to verify the installation:
    • node -v
  • This command checks the version of Node.js installed.
    • npm -v
  • This command checks the version of NPM installed.

    • 2. Installing Node.js and NPM on Linux

    Step 1: Update the System

  • Open a terminal window.
  • Update your package list by running:
    • sudo apt update

    Step 2: Install Node.js and NPM

  • On Debian-based distributions (like Ubuntu), you can install Node.js and NPM with the following command:
    • sudo apt install nodejs npm
  • This command installs both Node.js and NPM.

    • Step 3: Verify Installation

  • Run the following commands to verify that Node.js and NPM are installed correctly:
    • node -v
  • This checks the version of Node.js.
    • npm -v
  • This checks the version of NPM.

    • 3. Installing React.js Using create-react-app

    After installing Node.js and NPM, you can create a new React.js application using the create-react-app tool.

    Step 1: Install create-react-app

  • Open a terminal or command prompt and run:
    • npx create-react-app my-app
  • Replace my-app with the name you want for your project. This command will create a new directory with the project structure and install all necessary dependencies.

    • Step 2: Navigate to Your Project Directory

  • Change into the new project directory:
    • cd my-app

    Step 3: Start the Development Server

  • Start the React development server with:
    • npm start
  • This will launch your React application in your default web browser at http://localhost:3000/.

    • 4. Checking the Version of NPM

  • To check the version of NPM at any time, you can run:
    • npm -v
  • This command will display the current version of NPM installed on your system.
